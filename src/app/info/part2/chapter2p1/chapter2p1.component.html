Настройка вида таблицы

Настройка вида таблицы заключается в следующем:

  • Настройка колонок в таблице - применяется для конкретного рабочего места;
  • Сортировка записей в таблице - осуществляется для конкретного рабочего места в рамках одной сессии. После выхода из системы настройки сортировки сбрасываются;
  • Настройка оформления таблицы. Применяется для всех пользователей системы.
